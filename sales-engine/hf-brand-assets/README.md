# Q-GRID Comply — Brand Assets & Design System

Brand tokens, logo SVGs, and 3D configuration for the Q-GRID Comply platform.

## Contents

| File | Description |
|------|-------------|
| `brand-tokens.json` | Complete design token set (colors, fonts, spacing, radii) |
| `logo-concept-b.svg` | Selected logo — 3x3 Grid Mesh with Q tail |
| `logo-concept-a.svg` | Alternative — Crystalline Q hexagonal lattice |
| `logo-concept-c.svg` | Alternative — Quantum Orbital |
| `threejs-config.json` | Three.js 3D scene configuration for the interactive logo |

## Brand Summary

- **Primary**: `#00CCAA` (teal, dark mode) / `#008E8A` (deep teal, light mode)
- **Background**: `#0B0E14` (dark) / `#F8F9FA` (light)
- **Fonts**: DM Sans (body), Jura (headings), IBM Plex Mono (code/labels)
- **Radius**: 0px (landing), 8px (comply app)
- **Logo**: Concept B — 3x3 grid mesh + diagonal lattice bonds + Q tail
